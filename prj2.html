GoUsers Prototype v.1

Description

This project is a prototype for my startup GoUsers that I was working on during the summer. The purpose of this project is to create a software to enable users to help each other in IT subjects related to the website they are currently visiting. This project contained a server in Java using Servlets (J2EE for Tomcat) and running on Google Cloud platform. And a Google Chrome Extension client in JavaScript using Chrome's API. During this semester I continue the development of this project through a SJSU CS180H Honors class under the supervision of Prof. Ching-Seh (Mike) Wu aiming to produce optimal result of the crowdsourcing of help and collaboration within this software.
